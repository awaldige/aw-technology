## 🖥️ Sobre o Projeto

O **AW TECHNOLOGY** foi criado para oferecer uma experiência de compra fluida e moderna (UX/UI). O site é totalmente responsivo, adaptando-se a dispositivos móveis e desktops, com foco em performance e design minimalista.

### Principais Funcionalidades:
- **Catálogo Dinâmico:** Listagem de produtos com paginação e carregamento automático via JavaScript.
- **Carrinho de Compras:** Adição e remoção de itens com atualização de preço em tempo real.
- **Visualização de Senha:** Funcionalidade de alternar visibilidade no campo de login para melhor UX.
- **Checkout via WhatsApp:** Integração direta que envia o resumo do pedido para o vendedor.
- **Painel Administrativo:** Área restrita para cadastro, edição e exclusão de produtos.
- **Persistência de Dados:** Uso de `LocalStorage` para manter o estoque e o carrinho.
- **Níveis de Acesso:** Login diferenciado para Administrador (Edição total) e Convidado (Modo visualização).

## 🚀 Como testar o projeto

### Modo Cliente:
1. Acesse a página inicial.
2. Navegue pelos produtos e adicione itens ao carrinho.
3. Clique em "Finalizar no WhatsApp" para simular um pedido.

### Modo Administrativo (Painel de Controle):
1. Clique em **Painel ADM** no menu (ou acesse `login.html`).
2. Utilize as credenciais de demonstração:
   - **Chave de Acesso:** `convidado123`
3. No painel, você poderá cadastrar novos produtos.
